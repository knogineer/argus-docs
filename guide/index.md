# Getting Started

Welcome to the Argus Platform documentation! This guide will help you get started with development, deployment, and best practices.

## What is Argus?

Argus is a comprehensive intelligence platform built on Cloudflare Workers, providing:

- **Real-time collaboration** with Durable Objects
- **Multi-tenant architecture** with RBAC
- **GraphQL & REST APIs** for data access
- **WCAG 2.2 AAA accessibility** compliance
- **i18n support** (EN/ES/FR)

## Prerequisites

Before you begin, ensure you have:

- Node.js 18+ installed
- Python 3.12+ (for middleware)
- PostgreSQL access
- Neo4j access
- Cloudflare account
